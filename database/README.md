# Database

This folder contains the PostgreSQL database files for the Fraud Prevention System.

## Files

- `schema.sql` — creates the database tables
- `sample_data.sql` — inserts sample data for testing

## Tables

- merchant
- customer
- transactions
- fraud_check
- alert
- delivery_log
- audit_log

## Purpose

The database stores merchant data, customer data, transactions, fraud check results, fraud alerts, delivery logs, and audit logs.
