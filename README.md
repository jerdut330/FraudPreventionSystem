# Fraud Prevention System

An independent pre-gateway fraud prevention system for small e-commerce merchants.

## Project Description

This system helps merchants analyze suspicious transactions before payment processing. It calculates a fraud risk score, classifies transactions as low, medium, or high risk, and helps merchants review risky transactions before order fulfillment.

## Main Feature

- Transaction input
- Rule-based fraud scoring
- Risk classification
- Merchant fraud alert dashboard

## Tech Stack

- Frontend: React
- Backend: FastAPI
- Database: PostgreSQL
- Fraud Detection: Rule-Based Scoring
